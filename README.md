# DS4002_CS3

This repository contains the components necessary to complete the case study. This includes reference materials explaining the problem context and technical approach, as well as a hook document detailing the background and problem statement. A rubric detailing the specific expectations of the project is also included. 

## Important Information

The attached material in this repository detail the problem context and deliverables, as well as background on the problem. 

## Data

The dataset used for this project is sourced from Kaggle. Inside the Materials directory, a sub-directory named Data details the process for downloading the data from Kaggle and loading it in to Python, as well as provides an overview of the dataset itself.

## Materials

The `Materials' directory contains helpful resources for the completion of the case study. Within the Materials directory, the Technical directory contains a technical introduction to K-means clustering and TF-IDF vectorization, which are used in the modeling appraoch of the project. Within the Materials directory, the Context directory provides an article giving context to the problem, specifically detailing the issue of fake news detection by humans, and the potential applications of data science to this problem.
